# Privacy Policy

*Last Updated: January 30, 2026*

## 1. Data Collection
We minimize data collection to the absolute essentials required for service delivery:
- **GitHub Metadata:** Repository names, PR numbers, commit hashes.
- **Source Code:** Ephemerally processed for verification (AST Analysis, Math Validation). IT IS NOT STORED.
- **Payment Info:** Processed securely via GitHub Marketplace. We do not store credit card details.

## 2. Data Usage
- **Verification:** To generate pass/fail reports on your Pull Requests.
- **Improvements:** Aggregated, anonymized statistics (e.g., "10,000 math errors blocked") may be used to improve our engines.

## 3. Data Protection
- **Encryption:** All data in transit is encrypted via TLS 1.3.
- **Access Control:** Strict internal access controls based on the principle of least privilege.

## 4. Your Rights (GDPR/CCPA)
You have the right to request deletion of all metadata associated with your account. Contact [support@qwedai.com](mailto:support@qwedai.com).
