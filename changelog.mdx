## Upgrade Instructions

Follow these steps to upgrade your QWED-AI installation to the latest version.

<Steps>
  <Step title="Check Current Version" icon="package">

    Verify your installed version:

````bash
pip show qwed
````

  </Step>
  <Step title="Upgrade Package" icon="upload">

    Use the package manager for your environment:

    <CodeGroup tabs="pip,conda">
````bash
pip install --upgrade qwed
````

````bash
conda update qwed
````
    </CodeGroup>

  </Step>
  <Step title="Verify Installation" icon="check-circle">

    Test the upgrade:

````bash
qwed --version
qwed verify --help
````

  </Step>
  <Step title="Review Changelog" icon="book-open">

    Check release notes for breaking changes and migrate if needed.

  </Step>
</Steps>

<Callout kind="info">
  Always test in a staging environment before upgrading production deployments.
</Callout>
