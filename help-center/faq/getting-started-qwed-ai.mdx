## Overview

To start using QWED-AI, create a free account and set up your first documentation project.

## Steps

1. Visit `https://app.qwed.ai` and sign up
2. Click "New Project"
3. Name your project and choose a template

## Next

You can add pages, edit content, and invite collaborators from the dashboard.
